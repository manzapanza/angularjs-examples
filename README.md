# CODING DOJO JS #

## Instalação

    npm install -g gulp bower

    cd ~/Projects
    git clone https://github.com/manzapanza/coding-dojo-javascript.git
    cd coding-dojo-javascript/

    npm install && bower install

## Para rodar

    gulp serve
